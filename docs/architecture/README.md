# Архитектурные артефакты (ADD)

Артефакты процесса Attribute-Driven Design 3.0. Перекрёстные ссылки между файлами — см. схему в `.cursor/skills/add-process/templates.md`.

- **design-backlog.yaml** — драйверы и статусы Kanban (decision_ids, view_refs).
- **design-decisions.yaml** — решения, rationale, driver_id, view_ref.
- **views/** — эскизы видов (Mermaid, таблица ответственностей, секция Links).
- **inputs/** — опционально: design purpose, constraints, QA scenarios.
